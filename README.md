# 📊 MavenMarket Power BI Report

## 📖 Overview
This project is a **Power BI reporting system** for **Maven Market**, a global retail chain.  
The report contains **4 specialized dashboard pages** that monitor different aspects of the business, including:

- Financial performance
- Product targets
- Customer insights
- Regional and time-based trends

![Overview image of MavenMarket report](https://github.com/DragonTP/MavenMarket/blob/8da0679dd1767df07b555fff9b2fc64493f8719a/maven%20market%20project.jpg)

---

## 🛠️ Tech Stack

**Power BI**

Used for the entire workflow:

- Data processing  
- Data modeling  
- Dashboard visualization  

---

## ✨ Key Features

### 📊 Page 1: Overall Performance
Tracks the **main business KPIs**:

- **Total Revenue**
- **Profit**
- **Returns**

Visualizations include:

- Revenue **trend by month**
- Sales breakdown **by region**
- Performance comparison **by store type**

---

### 📦 Page 2: Product Performance

Focuses on **product-level targets and performance**.

Features include:

- **Gauge charts** comparing:
  - Revenue vs Target
  - Profit vs Target
  - Transactions vs Target

- **Price Adjustment Slider (What-if parameter)**: Allows users to simulate how **price changes impact profit**.

---

### 👥 Page 3: Customer Analysis

Manages and analyzes a customer database of **8,842 customers**.

Key metrics and segmentation:

- **Average Revenue Per Customer (ARPC)**
- Customer segmentation by:
  - Education level
  - Country
  - Occupation

---

### 🌍 Page 4: Geo & Time Analysis

Analyzes **regional and temporal sales patterns**.

Key insights include:

- Identifying **top-performing regions** using map visualizations
- Tracking **weekend transaction rates**
- Analyzing **revenue by day of the week**
